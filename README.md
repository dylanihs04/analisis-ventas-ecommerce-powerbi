# analisis-ventas-ecommerce-powerbi

📊 Análisis de Ventas E-Commerce

Proyecto de análisis de datos de un e-commerce utilizando SQL y Power BI para identificar patrones de ventas, comportamiento de clientes y rendimiento de vendedores.

El análisis se realizó utilizando el dataset Brazilian E-Commerce Public Dataset by Olist.

🛠️ Herramientas utilizadas
SQL
Microsoft Power BI
DAX
Modelado de datos
📊 KPIs principales

El dashboard incluye las siguientes métricas:

Total Revenue → ingresos totales
Total Orders → número total de pedidos
Total Customers → número total de clientes
Total Products Sold → productos vendidos
Average Order Value → valor promedio por pedido
📈 Análisis realizados

El dashboard permite analizar:

Evolución de ingresos en el tiempo
Categorías con mayores ventas
Top vendedores
Distribución de ventas por estado
Estado de los pedidos
